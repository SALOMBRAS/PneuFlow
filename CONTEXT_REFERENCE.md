# Cofre externo do PneuFlow

O contexto detalhado do projeto está em um repositório privado separado:

<https://github.com/SALOMBRAS/PneuFlow-Cofre>

O Cofre não deve ser consultado automaticamente.

Somente consultar quando o usuário solicitar expressamente:

- consulte o Cofre;
- atualize o Cofre;
- verifique a documentação arquitetural.

Quando autorizado:

1. abrir somente o `INDEX.md`;
2. abrir no máximo dois documentos diretamente relacionados;
3. não clonar nem pesquisar todo o Cofre;
4. não atualizar documentos sem autorização expressa.
